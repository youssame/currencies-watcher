# Currencies Watcher
It's a mobile app that watch currencies based on one based currency and presents their changes as Chart

**Requirements**
- Nodejs (Npm)
- React Native
- Expo

**Installation Steps**
 1. `git clone https://github.com/youssame/currencies-watcher.git` 
 2. `cd currencies-watcher` 
 3. `npm install` 
 4. `expo start`
